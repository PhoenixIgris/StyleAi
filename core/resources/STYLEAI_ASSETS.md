# StyleAI Asset Usage

The StyleAI SVG package is integrated through Compose Multiplatform resources in:

`src/commonMain/composeResources/drawable`

Use assets directly with `Res.drawable.<name>` or through `styleai.core.resources.StyleAiAssets`.

## Welcome Screen

- `ill_welcome_outfit_scene`: main welcome illustration
- `logo_styleai_wordmark`: optional brand mark
- `deco_sparkles`: light decorative accent

## Questionnaire Screen

- `ic_questionnaire_checklist`: questionnaire header
- `ic_outfit_hanger`: gender or style category
- `ic_occasion_calendar_star`: occasion
- `ic_color_palette`: skin tone or color palette
- `ic_face_shape`: face shape
- `ic_body_measurement`: height, fit, or comfort
- `ic_budget_price_tag`: budget
- `ic_work_briefcase`: work occasion
- `ic_wedding_ring`: wedding occasion
- `ic_date_heart`: date occasion
- `ic_party_discoball`: party occasion
- `ic_interview_person`: interview occasion

## Review Answers Screen

- `ic_edit_pencil`: edit action
- `ic_questionnaire_checklist`: answer summary
- `ic_color_palette`: palette summary
- `ic_body_measurement`: fit and height summary
- `ic_budget_price_tag`: budget summary

## Loading Screen

- `ill_ai_loading_cards`: main loading illustration
- `ic_ai_sparkle`: AI generation indicator
- `ic_magic_wand`: generation action
- `deco_soft_blobs`: subtle background decoration

## Recommendation Result Screen

- `ic_generate_refresh_sparkle`: regenerate action
- `ic_outfit_hanger`: outfit recommendation
- `ic_accessories_earrings`: accessories section
- `ic_shoes_footwear`: footwear section
- `ic_handbag`: bag section
- `ic_piggy_bank`: budget-friendly note

## Error Screen

- `ic_info_critical`: error state
- `ic_generate_refresh_sparkle`: retry action

## UI Tokens

- `color_palette_styleai`: visual color palette reference
- `logo_styleai_wordmark`: StyleAI wordmark
